# Zerg Pulse
Multiplayer web game, enter the website to inicialyze the game, or create a room website-url/room-name 

* ``W, A, S, D`` to move 

* ``M`` to use PULSE 

* ``K`` to use INVULNERABILITY 

### Prerequisites

NodeJS & npm

### Installing

Download or clone repo to local, execute console:
```
npm install
```
```
npm run build
```
```
npm start
```
