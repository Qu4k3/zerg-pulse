## Prerequisites

**Node.js** / **npm**

Check installation: *node --version* / *npm --version*


## Installing

Download or clone repo to local.
Once in the folder:

* Install packages defined in ``package.json``
```
npm install
```
* Start server // localhost:3000
```
npm start
```
